## Object Detection in Drone Dataset (contain UI)

### My Environment
- OS : Ubuntu 18.04.5
- GPU : NVIDIA RTX 2070 super
- CUDA : 10.2

### How to do?

```
$ git clone https://github.com/winston1214/Object_Detection_Drone && cd Object_Detection_Drone
```
```
$ pip install -r requirements.txt
```
```
$ python3 ui2.py
```
